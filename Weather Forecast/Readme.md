# 🌦️ Weather Dashboard
This is a simple and responsive weather app that lets you check real-time weather conditions for any city. It also shows a 5-day forecast and some useful highlights like humidity, wind speed, and pressure.
I built this project to practice working with APIs, handling user input, and improving UI using Tailwind CSS.

## 🚀 Features
- Search weather by city name  
- Get weather using your current location  
- Toggle temperature between °C and °F  
- 5-day forecast view  
- Weather highlights (humidity, wind, visibility, pressure)  
- Recent searches saved in browser  
- City suggestions while typing  
- Basic error handling for invalid inputs  

## 🛠️ Tech Stack
- HTML  
- Tailwind CSS  
- JavaScript  
- OpenWeatherMap API
