# Online Library System

A modern React-based Online Library System built using Vite, React Router DOM, Redux Toolkit, and Tailwind CSS.

## Features

### Home Page
- Welcome banner
- Book categories
- Popular books section
- Navigation bar

### Browse Books
- Browse books by category
- Dynamic routing (`/books/:category`)
- Search books by title or author
- Responsive book cards

### Book Details
- Detailed information for each book
- Author, description, rating
- Back to Browse button

### Add Book
- Add new books using a form
- Form validation
- Redux state management
- Automatic redirect after submission

### 404 Page
- Displays invalid URL
- Link back to Home
- Navbar hidden on 404 page

## Technologies Used

- React
- Vite
- React Router DOM
- Redux Toolkit
- Tailwind CSS

## Installation

### Clone Repository

git clone <your-repository-url>

### Navigate to Project

cd online-library

### Install Dependencies

npm install

### Run Development Server

npm run dev

### Build for Production

npm run build
