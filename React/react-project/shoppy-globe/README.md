# ShoppyGlobe E-Commerce Application

A React-based e-commerce application built using Vite, React Router, Redux Toolkit, and DummyJSON API.

## Features

- Product Listing
- Product Details Page
- Shopping Cart
- Quantity Management
- Redux State Management
- Search Products
- Checkout Page
- Order Placement
- Responsive Design
- Lazy Loading
- Custom Hooks
- Error Handling
- 404 Page

## Technologies Used

- React
- Vite
- Redux Toolkit
- React Router DOM
- React Hot Toast
- CSS3

## Installation

```bash
npm install
npm run dev
```
